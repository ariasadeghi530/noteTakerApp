# noteTakerApp

## About this app
- Neatly create and save notes in this simple app! 

## How to use this app
- On the notes page, simply create a note title, and add your note text below
- Click the save icon on the top right to save the note to the list
- Click on any note in the list to view its content
- Click the pencil icon to create a new note
- Click the trash icon to delete the note from the list
